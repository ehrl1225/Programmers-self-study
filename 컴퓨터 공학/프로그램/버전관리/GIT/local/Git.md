# gitignore 파일
Git이 추적하지 않아야 할 파일들을 지정하는 설정 파일
폴더 및 파일들을 적으면 git commit할 때 대상으로 뜨지 않는다.
따라서 git에 올라가면 안되는 파일들 예를 들어 api 키 파일, DB 계정 정보 같은 경우 포함시키지 않기 위해 사용한다.
