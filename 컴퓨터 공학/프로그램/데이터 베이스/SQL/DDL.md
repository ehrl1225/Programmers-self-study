# CREATE
데이터 베이스나 테이블을 생성할 때 사용한다.
예시)
```SQL
create table article(
	title VARCHAR(100),
    `body` TEXT
);
```


# ALTER
테이블의 구조를 변경할 때 사용한다.
예시)
```SQL
alter table article add column id int first;
```
