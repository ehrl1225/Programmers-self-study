
# 현재 창 닫기
## ctrl + w

# 이름 변경
f2

# 파일 탐색기
win + e

# 화면 좌우 분리
win + ← or →

# 화면 상하 분리
win + ↑ or ↓
# 바탕화면 보기
win + D

# 가상 테스크톱 만들기
ctrl + win + D

## 데스크톱간 이동
ctrl + win + ← or →

## 데스크톱 확인
win + tab

## 가상 데스크톱 삭제
ctrl + win + f4

# 작업 표시줄
win + 숫자키 1~0

## 새창으로 열기
win + shift + 숫자키

## 작업표시줄의 앱 선택
win + T

# 텍스트 블록 선택
shift + ctrl + ← or →

# 텍스트 줄 선택
shift + home or end

# 설정
win + I

# 화면 잠금
win + L

## 카톡 잠금
ctrl + L

### 줄 변경
shift + enter

# 인터넷 창 확대 축소
ctrl + 마우스 휠 위 아래
## 배율 100%
ctrl + 0

# 화면 캡쳐
win + shift + S

# 화면 고정
Power Toys를 설치해야함
ctrl + win + T
